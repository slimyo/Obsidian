# 解决匹配字符串时如何智能回溯问题

```cpp
P abcpabc S
|-------| |
 p|refix  |
  |-------|
     suffix 
```
- 前缀：字符串中以某字符开头的所有子串
- 后缀：字符串中以某字符结束的所有子串
- 相等前后缀：前缀字符串\=\=后缀字符串
- 最长相等前后缀子串：一个字符的子串串中，prefix\=\=suffix，中最长的那个

- 关键洞察：字符串增加一个字符后，其最长相等前后缀若增加，则最多为增加字符前的最长相等前后缀+1；
	- 则只要判断增加的这个字符i+1是否与上一个子串的最长相等前后缀长度j的后面（j+1）处字符是否相等
	- 不相等则回退：
		- 回退规则：
			- lps\[j-1\]处是已经匹配了的`pattern[0...j-1]`
			- 这一串前lps\[j-1\]和后lps\[j-1\]个字符相等
			- 当在位置 `j`失配时：判断lps\[j-1\]的后一处字符（注意：s\[lps\[j-1\]\]）是否与s\[i\]相等即可
			- 关键理解：**已匹配部分的最长公共前后缀**
				


- 算法
```cpp
void LPS(const string s,vector<int>* lps){
	int j = 0;//上一字符串最长相等前后缀长度
	// lps[0] = 0;
	for(int i=1;i<s.size();i++){
		// 此时j表示上一字符串最长相等前后缀长度+1（j-1不小于0）
		while(j>0&&s[i]!=s[j]){
			j = lps[j-1];//关键
		}
		if(s[i]==s[j]){
			j++;
		}
		lps[i] = j;
	}
}
```
